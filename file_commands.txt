help: вывести справку по доступным командам
info: вывести в стандартный поток вывода информацию о коллекции (тип, дата инициализации, количество элементов и т.д.)
show: вывести в стандартный поток вывода все элементы коллекции в строковом представлении
add <element>: добавить новый элемент в коллекцию
update <group_id>: обновить значение элемента коллекции, id которого равен заданному
remove_by_id <group_id>: удалить элемент из коллекции по его id
clear: очистить коллекцию
save: сохранить коллекцию в файл
execute_script <filename>: считать и исполнить скрипт из указанного файла.
exit: завершить программу
remove_first: удалить первый элемент из коллекции
head: вывести первый элемент коллекции
history: вывести последние 7 команд (без их аргументов)
max_by_expelled_students: вывести любой объект из коллекции, значение поля expelledStudents которого является максимальным
filter_by_group_admin <name;height;weight;country>: вывести элементы, значение поля groupAdmin которых равно заданному (значения вводить через ;), параметр country должен быть записан в верхнем регистре
filter_contains_name <name>: вывести элементы, значение поля name которых содержит заданную подстроку
